Interface AwsSigningConfigBase

Configuration for use in AWS-related signing. AwsSigningConfig is immutable. It is good practice to use a new config for each signature, or the date might get too old.

interface AwsSigningConfigBase {
    region: string;
    service?: string;
    date?: Date;
}
